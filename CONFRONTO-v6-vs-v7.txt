  🔧 IMPLEMENTAZIONE TECNICA
═══════════════════════════════════════════════════════════════

  v6.0 (Errato):
  ```css
  #calendario-iphone {
      max-width: 800px;  ← Limita TUTTI
      margin: 0 auto;
  }
  
  .cal-day {
      aspect-ratio: 1;  ← Problematico mobile
  }
  ```

  v7.0 (Corretto):
  ```css
  /* Default = Desktop GRANDE */
  .cal-day {
      min-height: 80px;
  }
  
  /* Solo mobile verticale = compatto */
  @media (max-width: 768px) and (orientation: portrait) {
      .cal-day {
          min-height: 42px !important;
      }
  }
  ```
